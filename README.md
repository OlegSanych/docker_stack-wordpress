# docker_stack-wordpress

## Cостав стека

Ubuntu 20.04 или Centos 8
CMS Wordpress (последняя(latest) версия)
PHP 7.4
MariaDB
Nginx
Cerbot
Redis
Phpmyadmin
Portainer
Docker
Docker-compose

Все устанавливаемые компоненты стека latest-версий.
